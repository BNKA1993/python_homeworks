1. Метод is_empty не обрабатывает строки с табуляцией как пустые

Описание:
Метод is_empty из класса StringUtils не распознает строки, содержащие только символы табуляции (\t), 
как пустые, что приводит к неверному результату в тестах.

Приоритет: Medium
Серьезность: S3

Шаги воспроизведения:
1.Создайте экземпляр класса StringUtils.
2.Вызовите метод is_empty с строкой, содержащей только символы табуляции (\t\t).
3.Проверьте результат.
Пример кода:
    def test_is_empty_with_tabulation():
        string_utils = StringUtils()
        res = string_utils.is_empty("\t\t")
        assert res == True

Ожидаемый результат: Метод is_empty должен распознавать строки, 
содержащие только пробельные символы (включая табуляцию), как пустые и возвращать True.

Фактический результат: Метод is_empty возвращает False для строки, содержащей только символы табуляции (\t\t).

Код ошибки: 
test_is_empty_with_tabulation ________________________________________________

    def test_is_empty_with_tabulation():
        string_utils = StringUtils()
        res = string_utils.is_empty("\t\t")
>       assert res == True
E       assert False == True
